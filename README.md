## What I'm Building
### 🔨 [AuxForge](https://auxforge.com) — AI Development Workbench
A desktop app where AI doesn't just generate code  you watch it work. Live terminal, live API responses, live database queries, live browser actions. Six integrated tools, one unified workspace.

**What I built solo:**
- **Electron desktop app** with multi-panel dock system
- **Real-time WebSocket relay** connecting IDE ↔ backend ↔ AI providers
- **Multi-agent orchestration** — 4 AI modes (Chat, Agent, Orchestrator, Follow AI)
- **Monaco code editor** with per-change accept/reject diffs
- **Browser automation** via Puppeteer with AI-driven page interaction
- **API tester, database explorer, terminal manager** — all AI-accessible
- **AI provider integration** — OpenAI, Claude, Gemini, Groq

### 🧠 Web Perception Engine
Fine-tuning **Vision modal** -> **1,000,000+ web UI annotations** to give AI agents structured perception of any webpage. Training runs distributed across AWS EC2 and GCP Compute Engine.
### 📚 UPSDMA-LMS
Full-stack learning management system with custom video player, multi-language audio sync, and segment-based progress tracking. Built for a government education initiative.

## The Story
Started as a side project called `sql_automa` in October 2024 — a tool that let AI interact with databases. Kept adding modules: terminal, API tester, file editor, browser control. Six months later, it wasn't a SQL tool anymore.
Registered **AuxForge Pvt Ltd**. Went full-time on building the development environment I wished existed.
When Anthropic's Claude Code source leaked (March 2026), the architecture was ~90% the same as what I'd already built independently. That felt like validation.
